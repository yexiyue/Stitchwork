# StitchWork 使用指南

## 应用简介

StitchWork 是一款服装加工流程管理系统，帮助服装加工作坊管理订单、计件和工资。

## 角色说明

### 老板（Boss）
工坊管理者，拥有全部管理权限：
- 管理订单和工序
- 录入计件记录
- 发放工资
- 管理员工和客户
- 查看统计数据

### 员工（Staff）
工坊工人，可以：
- 查看自己的计件记录
- 自助录入计件（需老板开启）
- 查看自己的工资记录

### 超级管理员（Admin）
平台管理员，管理注册码和用户。

## 业务流程

完整的加工流程：拿货 → 创建订单 → 添加工序 → 计件录入 → 出货 → 工资发放

## 老板功能指南

### 订单管理

#### 创建订单
1. 进入「订单」页面
2. 点击右上角「+」按钮
3. 填写订单信息：客户、货号、数量、备注
4. 点击「保存」

#### 添加工序
1. 进入订单详情页
2. 点击「添加工序」
3. 填写工序名称和计件单价
4. 可添加多个工序

#### 订单状态
- 进行中：正在加工
- 已完成：加工完成
- 已出货：已交付客户

#### 标记出货
在订单详情页点击「标记出货」，订单状态变为已出货。

### 计件录入

#### 录入计件
1. 进入「计件」页面
2. 点击「+」按钮
3. 选择订单和工序
4. 选择员工
5. 输入数量
6. 系统自动计算金额 = 数量 × 单价

#### 查看计件统计
在「计件」页面点击「统计」，可查看：
- 按员工统计
- 按订单统计
- 按时间段统计

### 工资管理

#### 查看工资汇总
进入「工资」页面，查看每个员工的：
- 计件总金额
- 已发放金额
- 待发放金额

#### 发放工资
1. 点击员工的工资记录
2. 点击「发放工资」
3. 输入发放金额
4. 确认发放

### 员工管理

#### 邀请员工
1. 进入「员工」页面
2. 点击「邀请员工」
3. 生成邀请二维码
4. 员工扫码加入工坊

#### 查看员工列表
在「员工」页面查看所有员工信息。

### 客户管理

#### 添加客户
1. 进入「客户」页面
2. 点击「+」按钮
3. 填写客户名称和联系方式

#### 查看客户订单
点击客户可查看该客户的所有订单。

### 数据分享

#### 创建分享链接
1. 进入「分享」页面
2. 点击「创建分享」
3. 选择分享内容（订单/计件/工资）
4. 设置有效期
5. 生成分享链接

分享链接可发送给客户或合作方查看数据。

## 员工功能指南

### 查看我的计件
进入「我的计件」页面，查看自己的所有计件记录。

### 自助录入计件
如果老板开启了自助录入功能：
1. 点击「+」按钮
2. 选择订单和工序
3. 输入数量
4. 提交等待老板审核

### 查看我的工资
进入「我的工资」页面，查看：
- 计件总金额
- 已发放金额
- 发放记录

### 计件统计
在「我的计件」页面点击「统计」，查看自己的计件汇总。

## 常见问题

### 如何注册账号？
老板注册需要注册码，联系平台管理员获取。员工通过扫描老板的邀请二维码加入。

### 忘记密码怎么办？
联系平台管理员重置密码。

### 如何修改个人信息？
进入「我的」页面，点击头像可修改昵称和头像。

### 计件金额计算错误怎么办？
检查工序单价是否正确，如需修改可在订单详情页编辑工序单价。

### 如何删除计件记录？
在计件详情页点击「删除」，仅老板有权限删除。

### 工资发放后能撤销吗？
已发放的工资记录不能撤销，请确认后再发放。

### 订单出货后还能修改吗？
已出货订单不能修改，如需调整请联系管理员。

### 如何查看历史数据？
在各列表页面可通过筛选条件查看历史数据。

### 数据安全吗？
所有数据加密存储，每个工坊数据相互隔离。

### 支持哪些设备？
支持 Android 手机和浏览器访问。
