# Add Process Management

## Summary

在订单详情页面中实现工序管理功能，支持工序的增删改查、搜索过滤和统计信息展示。

## Motivation

当前系统已有订单管理功能，但缺少工序管理的前端界面。工序是订单加工的核心环节，需要在订单详情中管理每个订单的工序，包括工序名称、计件单价等信息，并展示统计数据。

## Scope

- 订单详情页面重构，增加工序列表展示
- 工序 CRUD 操作（新增、编辑、删除）
- 工序搜索和过滤
- 工序统计信息（计件数量、金额）

## Out of Scope

- 独立的工序管理页面
- 工序模板功能
- 批量导入工序

## Dependencies

- 后端 API 已实现（`/api/processes`）
- 前端 API 客户端和 hooks 已实现
- 订单管理功能已完成
