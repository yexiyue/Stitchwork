# Proposal: add-stats-charts

## Summary

为老板端统计页面添加 ECharts 图表，通过柱状图和饼图直观展示员工产量数据。

## Motivation

当前统计页面仅以列表形式展示数据，缺乏直观的可视化图表。添加图表可以：
- 快速对比各员工产量差异
- 一目了然看到产量分布占比
- 提升数据分析效率

## Scope

### 老板端统计页面
- 安装 echarts + echarts-for-react
- 后端添加时间序列统计 API
- 柱状图（员工产量对比）
- 饼图（产量占比分布）
- 趋势折线图（按日产量变化）
- 支持数量/金额维度切换

### 员工端统计页面
- 创建员工统计页面入口
- 趋势折线图（个人按日产量）
- 堆叠柱状图（按订单/工序分组）
- 时间范围切换（本周/本月）

## Non-goals

- 不添加复杂的钻取分析功能

## Dependencies

- echarts: ^5.x
- echarts-for-react: ^3.x
