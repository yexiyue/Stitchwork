# Capability: workshop-settings

工坊自定义设置，允许老板配置计件单位和经营场所名称。

## ADDED Requirements

### Requirement: 计件单位设置

老板可以自定义计件单位名称，替代默认的「打」。

#### Scenario: 老板设置计件单位为「件」

**Given** 老板已登录且有工坊
**When** 老板在工坊设置页将计件单位改为「件」并保存
**Then** 系统保存设置，后续所有 UI 显示「件」而非「打」

#### Scenario: 新工坊使用默认单位

**Given** 老板创建新工坊
**When** 未设置计件单位
**Then** 系统使用默认值「打」

---

### Requirement: 场所名称设置

老板可以自定义经营场所的称呼，替代默认的「工坊」。

#### Scenario: 老板设置场所名称为「工厂」

**Given** 老板已登录且有工坊
**When** 老板在工坊设置页将场所名称改为「工厂」并保存
**Then** 系统保存设置，工坊设置页标题变为「工厂设置」

#### Scenario: 新工坊使用默认名称

**Given** 老板创建新工坊
**When** 未设置场所名称
**Then** 系统使用默认值「工坊」

---

### Requirement: 员工端显示老板设置

员工端 UI 应使用所属工坊的设置值显示单位和名称。

#### Scenario: 员工看到老板设置的单位

**Given** 老板设置计件单位为「双」
**And** 员工属于该工坊
**When** 员工查看计件记录
**Then** 数量显示为「{n}双」而非「{n}打」

---

### Requirement: 分享页显示工坊设置

公开分享页应使用工坊的设置值。

#### Scenario: 分享页显示自定义单位

**Given** 老板设置计件单位为「条」
**When** 访客查看招工分享页
**Then** 工序价格显示为「¥{n}/条」
**And** 剩余数量显示为「剩余{n}条」
