# Proposal: redesign-home-pages

## Summary

重新设计老板端和员工端首页，拆分为两个独立页面，添加数据概览、实时待审核徽章、迷你图表和最近动态功能。

## Motivation

当前首页 `_auth/index.tsx` 是一个共用页面，内容简单，缺乏：
- 数据概览（待审核数、订单统计、产量统计）
- 实时徽章显示待审核数量
- 可视化图表展示趋势
- 最近动态信息

重新设计可以提升用户体验，让老板和员工快速了解当前状态。

## Scope

### 后端 API

- 添加首页概览 API `GET /api/home/overview`
  - 老板：待审核数、进行中订单数、今日/本月产量
  - 员工：本月产量、本月金额
- 添加最近动态 API `GET /api/home/activities`
  - 返回最近的计件记录提交、审核等动态

### 老板端首页

- 工坊信息卡片（已有）
- 数据概览卡片（待审核、进行中订单、今日产量）
- 本周产量迷你趋势图
- 快捷操作入口
- 最近动态列表

### 员工端首页

- 工坊信息卡片（已有）
- 本月统计卡片（数量、金额）
- 本周产量迷你趋势图
- 快捷操作入口（录入计件）
- 最近记录列表

## Non-goals

- 不添加任务分配功能（保持自由录入模式）
- 不添加推送通知功能

## Dependencies

- add-stats-charts: 复用 ECharts 组件和 daily-stats API
